Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de compreender os fundamentos do funcionamento de aplicações web em Python sem o uso inicial de frameworks.
A aplicação utiliza:
  wsgiref.simple_server
  Manipulação manual de requisições HTTP
  Separação entre backend (Python) e frontend (HTML)
  Simulação simples de renderização dinâmica de conteúdo (template engine básica)
  O projeto servirá como base para evolução futura com o framework Flask, aplicando boas práticas e organização estrutural.
Objetivos de Aprendizado
  Entender o funcionamento do padrão WSGI
  Compreender como servidores web processam requisições
  Separar responsabilidades entre backend e frontend
  Manipular HTML dinamicamente com Python
  Preparar base para migração para Flask
Próximas Evoluções Planejadas
  Implementação de múltiplas rotas
  Melhor organização de pastas
  Uso de templates mais estruturados
  Migração para o framework Flask
  Implementação de formulários e métodos POST
  Integração com banco de dados
Evolução do Projeto
Este repositório representa o início da jornada no desenvolvimento web com Python, documentando a transição do entendimento básico de servidores WSGI até a construção de aplicações web mais completas com frameworks modernos.
